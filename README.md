# Classificador de Imagens - Gatos vs Cachorros
Projeto da disciplina de Robótica e Sistemas Inteligentes (Unidade 1).

## Tecnologias Utilizadas
* HTML5, CSS3, JavaScript
* TensorFlow.js
* Teachable Machine (Google)

## Como executar o projeto localmente
1. Faça o clone deste repositório ou baixe o ZIP.
2. **Aviso importante:** Devido a políticas de segurança de navegadores (CORS) ao carregar arquivos JSON locais no TensorFlow.js, não é possível abrir o arquivo `index.html` diretamente com um duplo clique.
3. Abra a pasta do projeto em um editor de código, como o **VS Code**.
4. Instale a extensão **Live Server** (caso não tenha).
5. Clique com o botão direito no arquivo `index.html` e selecione **"Open with Live Server"**. O projeto abrirá no seu navegador.
